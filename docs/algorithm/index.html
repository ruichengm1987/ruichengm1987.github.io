    1: leetcode题目按照什么顺序去刷

    按照一个框架来刷题目，比如说这周刷字符串,下周刷数组，按照一个章节一个章节这样来做，最好是跟着书、教程来一起做题. 做完教程的配套题目后leetcode上按频率从高到低刷,最少做每组的前20个。 做完一道题如果你比较陌生的思路， 继续做这道题的"相似问题”。面试题做一遍热题例如:https://leetcode-cn.com/problem-list/qg88wci/, https://leetcode-cn.com/problem-list/2cktkvj/

    参加周赛、双周赛，能做出前3个就不错了

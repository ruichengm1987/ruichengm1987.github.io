    国内镜像配置

    Go 1.13及以上(推荐)

    打开你的终端并执行:
    go env -w GOPROXY=https://goproxy.cn,direct

    Go module

    打开你的终端并执行:
    go env -w GO111MODULE=on

    go imports

    go imports包的引入可以让代码书写变得更加快捷
$ go get -v golang.org/x/tools/cmd/goimports

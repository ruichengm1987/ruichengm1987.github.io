    环境搭建

    安装

    1、 安装rustup
$ curl https://sh.rustup.rs -sSf | sh

2、执行
$ source $HOME/.cargo/env

    更新

    $ rustup update

    卸载Rust和rustup

    $ rustup self uninstall

    故障排除

    $ rustc --version  # 看到rustc x.y.z (abcabcabc yyyy-mm-dd)表示安装成功

    2.1.6.抽象类

    使用规则

    • abstract定义抽象类
    • abstract定义抽象方法,只有声明,不需要实现
    • 包含抽象方法的类是抽象类
    • 抽象类中可以包含普通的方法, 也可以没有抽象方法
    • 抽象类不能直接创建, 可以定义引用变量
    public abstract class Telphone {
    public abstract void call()
}

    5.双端队列

    特点：

    双端队列和普通队列最大的不同在于，它允许我们在队列的头尾两端都能在 O(1) 的时间内进行数据的查看、添加和删除。

    实现：

    与队列相似，我们可以利用一个双链表实现双端队列。

    应用场景：

    双端队列最常用的地方就是实现一个长度动态变化的窗口或者连续区间，而动态窗口这种数据结构在很多题目里都有运用。
